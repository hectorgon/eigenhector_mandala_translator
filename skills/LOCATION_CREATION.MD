# Location Creation Skill
---
description: A template and guide for documenting new locations, realms, and dimensions within the Federation Lore.
---

This skill provides a standardized format for documenting "Locations" or "Realms" discovered or created in the Federation. It integrates Magitech Survey data, Jataka Tales potential, and Ethical Simulation parameters.

## Location Documentation Format

Every location file should follow this structure:

# [Location Name]

## Lineage
- **Primary Source**: [Substack URL or Book Reference]
- **Associated Skills**: [e.g., @[skills/MAGITECH_SURVEY.MD], @[skills/lore/FEDERATION_TUTORIAL_REALM.MD]]

## Summary
A 2-3 paragraph description of the location's physical and metaphysical nature.

## Magitech Profile
- **Global Rank**: [Iron to Transcendent]
- **Technological Features**: [Engineering, Wordcel, Shape Rotator specializations]
- **Magical Prowess**: [Vibe Aura, Bridging, Resonance specializations]

## Inhabitants & Past Lives
Describe the typical beings found here.
- **Key Entity**: [Name/Description] (Link to Magitech Rank)
- **Avatar Potential**: How this location serves as a "past life" for the Avatar of Goodness, Truth, and Beauty.

## Ethics & Dharma-Sim (The Ethics of the Realm)
- **Primary Theme**: [e.g., Consent, Loyalty, Compassion]
- **Ethical Dilemma**: Describe a specific conflict heroes might face here, inferred from @[skills/lore/FEDERATION_ETHICS_SIM.MD].
- **Chiral Flip Potential**: What would a "negative" or "opposite" version of this realm's ethics look like?

## Tutorial Integration
- **Tutorial Realm Instance**: [Yes/No]
- **Scenario**: Description of the specific training scenario (e.g., Razor Tree) if applicable.
- **Protector in Residence**: [e.g., Kitty Garba, Cernunnos]

## Visual Synthesis (Thangka Prompt)
A detailed prompt for generating a visual representation of the realm.
- **Atmosphere**: [e.g., Shrouded, Luminous, Burning]
- **Symbols**: [Key items or geometries]
- **Style**: Thangka, Sacred Geometry, or specific artist style.

---
## Workflow for Creating a Location
1. **Gather Data**: Read the source material and related skills.
2. **Infer Ethics**: Use `FEDERATION_ETHICS_SIM.MD` to map the location's challenges to mythological or modern ethical themes.
3. **Draft the Profile**: Populate the sections above.
4. **Identify Protectors**: Assign Federation Protectors if the location is part of the Tutorial Realm.
5. **Generate Prompt**: Create the Thangka visual synthesis.
